-Another implementation of the iconic Swiss Railway Clock design; following the original design with just a few tweaks
-A straightforward analog watchface with a generous amount of code comments, much easier to read than any other sample you've seen
-A working implementation of on-device settings with a menu to set date display and dark mode options, which also uses
 a simple time picker
-A global settings class that synchronizes the selected options to persistent storage and makes them available across the app
-Compiles without warnings with the compiler type checking level set to "Strict"
